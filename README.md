## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Pravin-byte/MyPortfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the app locally:
   ```bash
   npm start
   ```
